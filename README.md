# 🎮 8-Bit Pixel Pet Game

一个纯前端的像素风宠物养成小游戏！

## 🐾 特性

- **4种像素宠物** — 猫咪、小狗、怪兽、兔子
- **3阶段成长系统** — 幼崽 → 青年 → 成年
- **宠物养成** — 喂食、清洁、玩耍、休息
- **猜数字小游戏** — 赢取快乐值
- **每日签到** — 连续签到领取奖励
- **8个成就解锁**
- **8-bit像素音效**
- **响应式设计** — 支持PC和手机

## 🎯 如何游玩

1. 注册账号并登录
2. 选择你的像素宠物
3. 照顾它，让它快乐成长！
4. 签到领取道具，完成成就

## 🛠 技术栈

- 纯 HTML + CSS + JavaScript
- 无需后端，localStorage 存储
- CSS 像素风设计
- Web Audio API 8-bit 音效
- 像素精灵图由 Python 脚本生成
